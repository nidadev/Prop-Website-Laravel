# Prop Website  Laravel Real Estate Platform

A full-featured real estate web application built with Laravel for browsing, searching, and connecting with property listings and real estate agents. This project simulates a US-style property marketplace with realtor contact, Google Maps integration, and Stripe payment support.

---

## 🚀 Features

- Property Listings (Buy / Rent)
- Advanced Property Search & Filters
- Realtor / Estate Agent Profiles
- Direct Contact with Agents (Call / Message)
- Google Maps Integration for property locations
- Stripe Payment Integration for premium listings
- User Authentication (Buyer / Seller / Admin roles)
- Favorite / Save Properties
- Responsive UI for all devices

---

## 🛠 Tech Stack

- Laravel (PHP Framework)
- MySQL
- REST API 
- Google Maps API
- Stripe Payment Gateway
- Bootstrap / Tailwind CSS
- JavaScript / AJAX
